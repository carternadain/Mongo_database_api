# Social-Network-API



# Description 
This repository encapsulates code created to use a NoSql database and 
get,post,delete in insomonia.



# Installation 
```
- npm Install (Should be all you need)
- npm install mongoose 
- npm install express 
- npm start to run app
```

# Technology:

- Node.js
- MongoDb
- Mongoose
- Javascript
- Node.js
